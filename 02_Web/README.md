# 学習の進め方

## 学習の進め方

1. 後述の学習テーマから好きなものを選んでIssueに登録する。
2. Issueのタイトルに合わせて、英字でフォルダを作成する。
3. ローカルにフォルダー名と同名でブランチを切る。
4. 学習スタート
